# Unity_Builds

## Space_Shooter for win
WASD for movement
Space- Fire
P - Pause
Escape - Main Menu
R - Restart
For 2nd player:
OKL; for movement (like WASD)
Right Shoft - fire

## Sci-Fi Demo for win
WASD for movement
E - interact ( you can pickup the coin, buy gun)
R - reload
Mouse0(left MB) - shoot
Note: box near you is destructible

## 2D Roguelike for win (Demo)
WASD - movement
Space - rotate "sword"
R - restart the Dungeon
Escape - Exit
